# 儿童皮疹决策指南

## 卡片1: 按压褪色 vs 不褪色
**标题**: 孩子身上的疹子按一下会消失吗
**内容**:
这是最重要的一步判断。用手指或透明玻璃杯按压疹子：
🟢 按压后颜色消失（褪色）：多数是普通病毒疹或过敏，不紧急。
🔴 按压后颜色不消失（不褪色，紫红色点/斑）：这是紫癜，可能提示脑膜炎或严重血液问题 → 立即急诊，不要等。
测试方法：用透明杯子压在疹子上，透过杯底观察颜色是否消失，比用手指更准确。

## 卡片2: 伴发烧的皮疹
**标题**: 孩子又发烧又出疹子
**内容**:
发烧+皮疹的组合需要重视，但不一定都很紧急。
🟡 先烧后疹（烧退了才出）：多数是幼儿急疹，3-5天自退，不需要特殊处理。
🟡 边烧边出疹、精神尚可：可能是普通病毒感染 → 今天去儿科门诊。
🔴 发烧+疹子按压不褪色 → 立即急诊。
🔴 发烧+皮疹+精神差/嗜睡/颈部僵硬 → 立即急诊，警惕脑膜炎。
参考自儿科急诊分级方法论。

## 卡片3: 不伴发烧的皮疹
**标题**: 孩子没发烧但身上起了疹子
**内容**:
没有发烧的皮疹多数不紧急，但要注意几个例外。
🟢 小片红疹、不痒或轻微痒、精神好 → 观察1-2天，可能自行消退。
🟢 接触新食物/新衣物后出现 → 可能是接触性过敏，去除过敏原后观察。
🟡 大面积荨麻疹（红色风团，此起彼伏）→ 可口服抗过敏药，不好转则今天去门诊。
🔴 皮疹+不褪色紫癜，即使不发烧 → 今天必须就医。
参考自儿科急诊分级方法论。

## 卡片4: 伴呼吸困难或嘴唇肿
**标题**: 孩子出疹子同时呼吸不好或嘴肿了
**内容**:
🔴 这是严重过敏反应（过敏性休克）的信号，必须立即处理！
皮疹/荨麻疹 + 以下任何一项 → 立即拨打120：
- 呼吸困难、喘息、喉咙发紧
- 嘴唇/舌头/眼睛肿胀
- 面色苍白或嘴唇发紫
- 头晕、站不稳、意识模糊
如果有医生开过肾上腺素笔（如 EpiPen），立即使用，然后仍然要拨120。
等待救护车时让孩子平躺、抬高双腿、不要喂食喂水。

## 卡片5: 水痘识别和家庭隔离
**标题**: 怎么判断是不是水痘、怎么护理
**内容**:
水痘典型特征：先发烧，然后出现红色小丘疹 → 变成透明水疱（像露珠）→ 水疱浑浊结痂。新旧皮疹同时存在，头皮和躯干最多。
🟡 确认水痘后：居家隔离至所有水疱完全结痂（通常7-10天），不要去学校。
护理要点：剪短指甲防抓破感染、穿宽松棉质衣服、可用炉甘石洗剂止痒。
🔴 水痘+高热不退/疱疹化脓/精神极差 → 今天去医院。
参考自儿科急诊分级方法论。

## 卡片6: 手足口病识别和家庭护理
**标题**: 手足口病怎么认、怎么护理
**内容**:
手足口病典型特征：口腔内疼痛性溃疡 + 手掌、脚掌、臀部出现小水疱或红疹。多见于5岁以下儿童，夏秋季高发。
🟡 轻症（低热+吃得下+精神好）：居家隔离2周，主要护理是缓解口腔疼痛——可吃凉的软食（冰酸奶、凉粥），避免酸辣食物。
🔴 必须立即就医的信号：持续高热>39℃超过2天、频繁呕吐、手脚抖动或无力、嗜睡叫不醒、呼吸急促 → 这些提示可能发展为重症，立即急诊。
参考自儿科急诊分级方法论。
