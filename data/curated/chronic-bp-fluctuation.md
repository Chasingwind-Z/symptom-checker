# 血压波动管理决策指南

## 卡片1: 血压突然升高怎么办
**标题**: 量血压发现160/100以上了
**内容**:
先别慌，坐下来休息10分钟后重新量一次，很多"高血压"是刚运动、生气、喝咖啡后的临时升高。
复测仍然>180/110 + 没有任何不舒服 → 按时吃降压药，1-2小时后复测，不需要跑急诊。
复测>180/110 + 有头痛头晕恶心 → 今天去门诊或急诊调药。
🔴 紧急：血压>180/120 + 剧烈头痛+视物模糊+呕吐+胸痛 → 立即拨120，可能是高血压急症。
一次高不代表需要加药，连续多天高才需要调整方案。

## 卡片2: 血压偏低头晕
**标题**: 量血压只有90/60，头发晕
**内容**:
低血压标准：收缩压<90 或 舒张压<60 mmHg。
如果平时血压就偏低（比如100/60）且没有不舒服 → 不需要处理，这是你的正常水平。
如果平时120/80突然降到90/60 + 头晕眼花 → 可能是降压药过量、脱水或其他原因。
紧急处理：先躺下抬高双腿、喝温盐水（一杯水+半茶匙盐）。不要自行停降压药但今天联系医生。
🔴 低血压+意识模糊+出冷汗 → 立即拨120。

## 卡片3: 晨起血压高峰
**标题**: 每天早上血压特别高
**内容**:
"晨峰高血压"是很常见的现象：清晨6-10点是血压自然高峰期。
如果只是早上偏高（比如145/90）、其他时间正常 → 跟医生说，可能需要调整吃药时间或换长效药。
晨起后正确测量方法：起床后排尿、坐5分钟、测2次取平均值。
如果晨起血压经常>160/100 → 这周去找医生调药，不要自己加量。
有些医生会建议睡前吃一次长效降压药覆盖晨峰，但这需要医生评估决定。

## 卡片4: 忘记吃降压药了
**标题**: 降压药忘吃了怎么补
**内容**:
发现忘吃时距离正常时间<12小时 → 马上补吃。
距离正常时间>12小时或快到下一次吃药时间 → 不补了，下次正常吃，不要加倍。
一天吃2次的药漏了一次 → 想起来就补，但两次间隔不少于4小时。
偶尔漏1次不会出大问题，但连续漏2天以上血压可能反弹性升高，比较危险。
⚠️ 建议设手机闹钟提醒吃药，或用分药盒每周装好。

## 卡片5: 血压测量注意事项
**标题**: 在家量血压为什么每次都不一样
**内容**:
血压本身就是波动的，一天内变化20-30mmHg是正常的。
正确测量：每天固定时间（建议早晚各一次）、排尿后、安静坐5分钟、袖带绑在心脏同高的上臂。
常见误差原因：刚运动、说话、腿交叉、袖带太松太紧、手机看到让你紧张的消息。
每次测2遍取平均值，记录下来。连续7天的平均值才是你真正的血压水平。
家用血压计建议每年去医院校准一次。

## 卡片6: 降压药常见问题
**标题**: 吃降压药的一些常见困惑
**内容**:
"血压正常了能停药吗" → 大多数不能。血压正常是因为药在起作用，停了会反弹。
"吃药吃到什么时候" → 高血压目前需要长期控制，就像近视需要一直戴眼镜。
"降压药有副作用怎么办" → 告诉医生具体症状（干咳、脚肿、头晕等），换药不要停药。
ACEI类（普利）常见干咳 → 可以换成ARB类（沙坦）。
钙拮抗剂（地平）可能脚踝水肿 → 加小剂量利尿剂或换药。

## 卡片7: 血压与日常生活
**标题**: 高血压患者生活中要注意什么
**内容**:
限盐是最有效的生活方式干预：每天<5g盐（大约一啤酒瓶盖）。注意"隐形盐"：酱油、腌菜、方便面、加工食品。
适量运动：每周至少150分钟中等强度运动（快走、游泳），但血压>180时不要剧烈运动。
限酒：男性每天白酒不超过1两、啤酒不超过1瓶；女性减半。最好不喝。
减重：每减5公斤，收缩压大约降低4-5mmHg。
⚠️ 不要用"养生偏方"代替降压药，喝醋、泡脚不能治高血压。

## 卡片8: 血压波动与情绪
**标题**: 一生气血压就飙高
**内容**:
情绪激动确实会让血压短时飙升20-40mmHg，这是正常的应激反应。
生气/紧张时测到高血压 → 先深呼吸、离开刺激环境、15分钟后再测。
如果经常因为情绪波动导致血压忽高忽低 → 跟医生讨论是否需要调药或加用抗焦虑治疗。
"白大衣高血压"：去医院就紧张血压就高，在家正常 → 建议做24小时动态血压监测确认。
长期情绪不好的人：建议同时关注心理健康，焦虑抑郁本身也是心血管危险因素。

## 卡片9: 血压药与其他药的相互影响
**标题**: 降压药能不能跟其他药一起吃
**内容**:
感冒药（含伪麻黄碱）：会升高血压，高血压患者选不含"伪麻黄碱"的感冒药。
止痛药（布洛芬等NSAIDs）：长期使用会升高血压、降低降压药效果。偶尔吃1-2天没大问题。
降糖药：有些降压药（比如ACEI/ARB）对糖尿病有保护作用，不冲突。但β受体阻滞剂可能掩盖低血糖症状。
中成药/保健品：很多含有升压成分（甘草、麻黄等），吃之前最好问医生。
⚠️ 每次看任何科的医生，都要告诉他你在吃降压药。

## 卡片10: 什么时候血压波动不用太担心
**标题**: 血压不是完全稳定的，这些波动是正常的
**内容**:
一天之内波动10-20mmHg → 正常的生理节律，白天高晚上低。
季节变化：冬天血压偏高、夏天偏低，差值5-10mmHg → 正常，但冬天注意监测。
运动时血压升高、运动后降低 → 这是正常反应。
偶尔一次测到偏高但平时都正常 → 不用紧张，可能是测量误差或临时因素。
关键看趋势：连续一周的平均值>140/90才考虑调方案，不要被单次数值吓到。
