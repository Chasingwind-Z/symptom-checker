# 老年人跌倒决策指南

## 卡片1: 跌倒后第一时间评估
**标题**: 老人摔倒了，先别急着拉起来
**内容**:
🔴 最重要的一条：不要立刻把老人拉起来！先在地上做评估。
先问老人：哪里疼？能动吗？头有没有磕到？
如果清醒+四肢能动+没有明显疼痛 → 让老人慢慢侧身，用手撑起，先坐一会再站起。
如果意识模糊、说不清话、四肢不能动、颈部/背部/髋部剧痛 → 不要搬动，立即拨120。
⚠️ 强行拉起可能加重骨折移位或脊髓损伤。

## 卡片2: 头部着地的观察
**标题**: 老人摔倒磕到头了
**内容**:
磕到头即使当时看着没事，也要观察48小时。老年人硬膜下出血可能延迟出现。
必须去急诊的情况：头上有明显肿包或出血、呕吐超过1次、头痛越来越重、一侧手脚变得没力气、说话含糊、嗜睡叫不醒。
正在吃抗凝药（华法林、利伐沙班等）的老人磕到头 → 无论有无症状都必须今天去急诊做CT。
观察期间：每2小时叫醒老人一次（如果是睡觉时间），确认意识清楚。

## 卡片3: 髋部骨折判断
**标题**: 老人摔倒后说胯骨疼
**内容**:
老年人跌倒最怕髋部（大腿根部/胯部）骨折，骨质疏松者轻轻一摔就可能骨折。
怀疑髋骨折的信号：摔倒后站不起来、一侧腿不能抬起、受伤那条腿看着比另一条短或外旋。
确认方法：让老人平躺，尝试轻轻抬起受伤腿的脚后跟（不要强行），如果完全抬不起+剧痛 → 高度怀疑骨折。
处理：不要搬动患肢，在地上垫上毯子保暖，拨打120。

## 卡片4: 跌倒后腕部和肩部受伤
**标题**: 老人摔倒手撑地后手腕肿了
**内容**:
用手撑地摔倒最容易伤手腕（桡骨远端骨折），表现为手腕肿胀、变形、不能握东西。
肩膀着地可能造成肩关节脱位或肱骨骨折：手臂不能抬起、肩膀外形异常。
今天处理：用围巾或布条做简易固定悬吊在胸前，冰敷15分钟，今天去骨科。
不要自己尝试"掰回去"或者用力活动。
如果手指能动、血色正常 → 不算最紧急，但今天内必须看医生。

## 卡片5: 跌倒后没有明显外伤
**标题**: 老人摔了一下但说没事
**内容**:
"说没事"不代表真的没事。老年人痛觉敏感度下降，可能骨折了也觉得"就是有点疼"。
接下来24-48小时注意观察：能不能正常走路、疼痛是加重还是减轻、有没有新出现的淤青或肿胀。
如果第二天起床后发现摔的地方更疼了、活动受限 → 今天去拍片。
无外伤但连续摔倒2次以上 → 必须去查原因（血压、心脏、药物、视力、平衡功能）。

## 卡片6: 跌倒原因要当心
**标题**: 老人为什么会摔倒，比摔伤更重要
**内容**:
被东西绊倒 → 环境问题，整理家里（见卡片10预防）。
站起来时头晕摔倒 → 可能是体位性低血压，测卧位和站位血压，3天内看内科。
走路时突然腿软 → 可能是脑血管问题，尽快去神经内科。
无缘无故摔倒+短暂意识丧失 → 可能是心律失常或短暂脑缺血，今天去急诊做心电图和脑CT。
🔴 跌倒前有胸闷、心慌、黑矇 → 现在去急诊。

## 卡片7: 服用特殊药物的老人跌倒
**标题**: 吃降压药/安定的老人容易摔倒
**内容**:
高风险药物：降压药（尤其刚调量）、安眠药/镇静药、降糖药（低血糖）、利尿剂（头晕）、止痛药（头晕嗜睡）。
如果最近刚换药或加药后开始容易摔 → 不要自己停药，但这周去找开药的医生说明情况。
吃安眠药的老人：起夜时摔倒风险最高，建议床边放夜灯和尿壶。
降压药在睡前吃的老人：夜间血压可能过低，起夜时一定慢起来（先坐30秒再站）。

## 卡片8: 骨质疏松与反复跌倒
**标题**: 老人动不动就骨折
**内容**:
如果轻微跌倒就骨折 → 强烈提示骨质疏松，需要做骨密度检查。
骨质疏松不只是补钙，需要医生制定治疗方案（可能需要抗骨松药物）。
补充维生素D：很多老人维生素D不足，建议查25-OH维生素D水平。
日常补钙量：60岁以上建议每天1000mg钙+800IU维生素D（食物+补充剂合计）。
已有骨折史的老人 → 再次骨折风险极高，必须启动正规骨松治疗。

## 卡片9: 跌倒后心理影响
**标题**: 摔过一次后老人不敢走路了
**内容**:
"恐跌症"很常见：摔过一次后害怕再摔，减少活动，反而导致肌肉萎缩、更容易摔。
恶性循环：摔倒 → 不敢动 → 肌力下降 → 更容易摔。
家属要做的：鼓励老人继续活动、陪伴行走给予安全感、考虑使用拐杖或助行器。
如果老人明显变得不愿出门、情绪低落 → 可能需要同时关注心理健康。
建议老人做简单的平衡训练：扶着椅子单腿站立、太极拳、慢走。

## 卡片10: 家庭防跌倒措施
**标题**: 怎么让家里更安全防止老人摔倒
**内容**:
卫生间：安装扶手（马桶旁+淋浴区）、使用防滑垫、夜灯常亮。
卧室：床高不要太高（坐在床沿脚能平放地面）、床头放手电和手机、拖鞋换成防滑鞋。
客厅/走廊：移走地毯或用防滑底的、电线归拢贴墙、保持通道畅通。
台阶/门槛：贴防滑条、标明显颜色、有条件装小斜坡。
穿着：不穿太长的裤子、选择合脚的防滑鞋（不要穿拖鞋外出）。
照明：全屋保证足够亮度，走廊和楼梯装感应灯。
