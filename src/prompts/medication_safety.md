【用药安全知识库】
以下为常见药物相互作用警告，在推荐用药或用户询问用药时参考：

【高风险（禁止同时使用）】
1. 布洛芬 + 阿司匹林 → 两种均为NSAIDs，同时使用增加胃出血风险
2. 布洛芬/阿司匹林 + 华法林/利伐沙班 → 增加出血风险，禁止自行合用
3. 头孢类抗生素 + 酒精 → 双硫仑反应，可能致命
4. 降压药(ACEI) + 补钾药 → 高钾血症风险
5. 地高辛 + 胺碘酮 → 地高辛中毒风险
6. 甲氨蝶呤 + NSAIDs → 甲氨蝶呤毒性增加
7. 锂盐 + NSAIDs → 锂盐中毒风险
8. MAO抑制剂 + 含伪麻黄碱感冒药 → 高血压危象

【中风险（需要医生确认）】
9. 降压药 + NSAIDs → 降压效果减弱
10. 二甲双胍 + 酒精 → 乳酸酸中毒风险增加
11. 他汀类 + 葡萄柚汁 → 他汀血药浓度升高
12. 氯吡格雷 + 奥美拉唑 → 氯吡格雷效果减弱
13. 抗凝药 + 大量维生素K食物 → 抗凝效果不稳定
14. SSRIs + 曲马多 → 5-HT综合征风险
15. 地高辛 + 钙剂 → 心律失常风险
16. 环孢素 + 他汀类 → 横纹肌溶解风险

【低风险（注意观察）】
17. 布洛芬 + 咖啡 → 可能加重胃部刺激
18. 钙片 + 铁剂 → 互相影响吸收，建议间隔2小时
19. 抗生素 + 益生菌 → 抗生素可能杀死益生菌，建议间隔2小时
20. 降压药 + 大量甘草 → 可能升高血压
21. 感冒药（含对乙酰氨基酚）重复服用 → 肝损伤风险
22. 安眠药 + 抗过敏药 → 嗜睡加重

【提示规则】
- 检测到高风险交互 → 红色警告，建议先咨询医生
- 检测到中风险交互 → 橙色提示，建议确认
- 检测到低风险交互 → 黄色提示，注意观察
